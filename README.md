# Giddaa Coding Interview

## Run Locally

Clone the project

```bash
  git clone https://github.com/BoltUltra/giddaa.git
```

Go to the project directory

```bash
  cd giddaa
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Open in browser

```bash
  http://localhost:5173/
```
